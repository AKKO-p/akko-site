Platform

Not a bundle of tools.
A platform that works.

26 services, deeply integrated. Unified SSO, automatic lineage, pre-configured monitoring. Replace any layer without touching the others.

Your Data Stays Portable

Apache Iceberg with ACID transactions, time-travel, schema evolution. Polaris catalog with RBAC. Stored on MinIO. No format lock-in. Ever.

Query Anything, Anywhere

Spark for heavy compute. Trino for federated SQL across Iceberg and PostgreSQL. Airflow for orchestration with OpenLineage tracking every step.

AI That Never Phones Home

Ollama runs LLMs locally. pgvector stores RAG embeddings. jupyter-ai for conversational AI. Your data never leaves your infrastructure. Period.

Know Where Every Byte Goes

OpenMetadata for catalog, lineage, PII classification, glossary, data quality tests. Domains, data products, ownership — automated from day one.

One Login. Every Service.

Keycloak SSO with 14 pre-configured OAuth clients. 4 RBAC roles. Trino row-level access control. Authenticate once, access everything.

See Everything. Miss Nothing.

Prometheus metrics. Grafana dashboards. Loki logs. AlertManager alerts. Full observability for every service, pre-configured and ready.

Code in Any Language

JupyterHub with Python, R, Julia, VS Code, and Quarto. Pre-installed libraries. Publish reports. code-server built in for full IDE experience.

Dashboards That Build Themselves

Apache Superset with auto-provisioned datasets, charts, and dashboards. SQL Lab for ad-hoc queries. Connected to Iceberg and PostgreSQL through Trino.

Get Started

Two commands.
Twenty-six services. Ready.

All you need is Docker Desktop with 6 GB RAM. That's it. No Kubernetes. No Terraform. No cloud account.

1

Clone and launch

git clone https://github.com/AKKO-p/AKKO && cd AKKO ./scripts/start.sh
2

Open the Cockpit

Navigate to https://localhost. All 26 services visible in one portal. Live health checks. One-click access to everything.

3

Run the demo

Open JupyterHub. Run the banking notebook. It creates Iceberg tables via Spark, queries them with Trino, and provisions a Superset dashboard. End to end. One click.

Prerequisites

Docker Desktop (6 GB+ RAM) Docker Compose v2+ 20 GB free disk macOS or Linux
