Capabilities Fonctionnalités

Where AI, engineering,
and analytics converge.

Là où l'IA, l'engineering
et l'analytique convergent.

30+ next-generation services, deeply integrated. A self-hosted open source lakehouse with sovereign AI, governance, monitoring, and a full IDE — all connected. Replace any layer without touching the others.

Plus de 30 services dernière génération, profondément intégrés. Un lakehouse open source auto-hébergé avec IA souveraine, gouvernance, supervision et un IDE complet — tout est connecté. Remplacez n'importe quelle couche sans toucher aux autres.

Your Data Stays Portable

Apache Iceberg (not Hive) with ACID transactions, time-travel, schema evolution. Apache Polaris catalog (not Hive Metastore) with RBAC. Stored on MinIO. No format lock-in. Ever.

Query Anything, Anywhere

Spark Connect for heavy compute (not legacy Spark). Trino for federated SQL across Iceberg and PostgreSQL. Airflow for orchestration with OpenLineage tracking every step. All state-of-the-art 2026 engines.

Sovereign AI, Built Into Your Notebooks

Ollama runs LLMs locally (Qwen 2.5-Coder:7B, Qwen 2.5:3B) with nomic-embed-text for embeddings. LiteLLM provides an OpenAI-compatible API gateway routing to any model. MCP servers enable tool calling. pgvector stores RAG embeddings in PostgreSQL. jupyter-ai adds a conversational AI assistant directly inside your notebooks. LangChain pipelines ready to go. Zero calls to OpenAI, Google, or Anthropic. Your data stays yours.

Know Where Every Byte Goes

OpenMetadata for catalog, lineage, PII classification, glossary, data quality tests. Domains, data products, ownership — automated from day one.

One Login. Every Service.

Keycloak SSO with 14 pre-configured OAuth clients. OPA (Open Policy Agent) for fine-grained authorization — row-level security, column masking, policy-as-code RBAC. Trino table-level access control. Authenticate once, access everything.

See Everything. Miss Nothing.

Prometheus metrics. Grafana dashboards. Loki logs. AlertManager alerts. Full observability for every service, pre-configured and ready.

A Professional IDE in Your Browser

code-server (VS Code) with 20 pre-installed extensions — Python, Jupyter, SQLTools, dbt Power User, Git Lens, REST Client, and more. JupyterHub with Python, R (IRkernel), and Julia (IJulia) kernels. Quarto for publishing reports as HTML, PDF, or full websites. No local install. Open a tab, start coding.

Dashboards That Build Themselves

Apache Superset with auto-provisioned datasets, charts, and dashboards. SQL Lab for ad-hoc queries. Connected to Iceberg and PostgreSQL through Trino.

AI-Native Cockpit

A control plane with built-in AI chat. Ask questions about your data catalog, query tables, check service health — all from the cockpit. Powered by LiteLLM and MCP servers for tool calling.

Vos données restent portables

Apache Iceberg (pas Hive) avec transactions ACID, voyage dans le temps, évolution de schéma. Catalogue Apache Polaris (pas Hive Metastore) avec RBAC. Stockées sur MinIO. Aucun verrouillage de format. Jamais.

Interrogez tout, partout

Spark Connect pour le calcul intensif (pas du Spark legacy). Trino pour le SQL fédéré entre Iceberg et PostgreSQL. Airflow pour l'orchestration avec OpenLineage qui trace chaque étape. Des moteurs state-of-the-art 2026.

IA souveraine, intégrée dans vos notebooks

Ollama fait tourner les LLM en local (Qwen 2.5-Coder:7B, Qwen 2.5:3B) avec nomic-embed-text pour les embeddings. LiteLLM fournit une passerelle API compatible OpenAI vers n'importe quel modèle. Les serveurs MCP activent l'appel d'outils. pgvector stocke les embeddings RAG dans PostgreSQL. jupyter-ai ajoute un assistant IA conversationnel directement dans vos notebooks. Pipelines LangChain prêts à l'emploi. Zéro appel vers OpenAI, Google ou Anthropic. Vos données restent les vôtres.

Sachez où va chaque octet

OpenMetadata pour le catalogue, le lignage, la classification PII, le glossaire et les tests de qualité. Domaines, produits de données, propriétaires — automatisés dès le premier jour.

Un seul identifiant. Tous les services.

SSO Keycloak avec 14 clients OAuth préconfigurés. OPA (Open Policy Agent) pour l'autorisation fine — sécurité par ligne, masquage de colonnes, RBAC policy-as-code. Contrôle d'accès Trino par table. Connectez-vous une fois, accédez à tout.

Tout voir. Ne rien manquer.

Métriques Prometheus. Tableaux de bord Grafana. Logs Loki. Alertes AlertManager. Observabilité complète pour chaque service, préconfigurée et prête.

Un IDE professionnel dans votre navigateur

code-server (VS Code) avec 20 extensions préinstallées — Python, Jupyter, SQLTools, dbt Power User, Git Lens, REST Client, et bien plus. JupyterHub avec les kernels Python, R (IRkernel) et Julia (IJulia). Quarto pour publier vos rapports en HTML, PDF ou sites web complets. Aucune installation locale. Ouvrez un onglet, codez.

Des tableaux de bord qui se construisent seuls

Apache Superset avec jeux de données, graphiques et tableaux de bord autoprovisionnés. SQL Lab pour les requêtes ad hoc. Connecté à Iceberg et PostgreSQL via Trino.

Cockpit IA-natif

Un panneau de contrôle avec chat IA intégré. Interrogez votre catalogue de données, requêtez vos tables, vérifiez l'état des services — tout depuis le cockpit. Propulsé par LiteLLM et les serveurs MCP pour l'appel d'outils.
